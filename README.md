# Cpitt's Dotfile

Dotfiles managed using a bare git repository

## Installation:
```
curl -Lks https://raw.githubusercontent.com/cpitt/dotfiles2/master/.dotfiles-install.sh | /bin/bash
```

## Updating dotfile

```
config pull
```

## Adding files
```
config add .newFile
config commit -m 'adding .newFile'
config push
```
